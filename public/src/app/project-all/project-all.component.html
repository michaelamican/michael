Projects

  • Illuminare Studio

    Illuminare Studio is a private leather distributor based in Los Angeles. Michael was the sole contributor and full stack developer for this project, and built the website from scratch. The website is built with a Django backend, and hosted on AWS EC2 with Route 53. Front-end technologies used are Bootstrap, jQuery, Javascript, and CSS Grid for the layout. The website is media-responsive, built for mobile, iPad, and desktop. It features slideshows and modals, as well as custom UI.

  • Community Care and Hospice

    Community Care and Hospice is a hospice in Camarillo, CA. Michael was one on a team of two. The project is a Django backend with jQuery, Javascript, and CSS Grid. Michael was pulled in as a front-end designer to add media responsive functionality. Much of the project involved removing in-line styling and condensing 70+ CSS files into 6 component-based CSS files. Michael also built custom navigation UI, and added accessibility features.

  • Brown Paper LA

    Brown Paper LA is a local catering company who already had a website running, but needed a facelift. Per client request, Michael kept the website hosted on Squarespace, and transferred the menu ordering functionality to the Squarespace site. Michael connected Stripe, built an ordering form, created a menu, edited all of their image assets, and added purchase functionality. Michael also completely redesigned the color and layout scheme.
